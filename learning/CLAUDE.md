# CLAUDE.md — LangChain 学习项目配置

## 项目定位

这是一个 LangChain 源码学习项目，不是生产开发项目。以**理解概念、动手实践**为目标，不追求代码完整性。

## 学习资源优先级

按以下顺序查阅，不要一上来就翻源码：

1. **官方文档**（通过 MCP 工具查询，已配置在 `.mcp.json`）
2. **课程大纲**（`learning/课程大纲.md` — 每节课指定了对应的文档页）
3. **学习指南**（`learning/学习指南.md` — 源码阅读索引，按需查阅）
4. **源码**（`libs/` — 带着具体问题看，不要通读）

## 工作目录

所有练习代码放在 `learning/demos/` 下，命名规范：`lesson01_xxx.py`

## 开发环境

```bash
# 进入 core 包目录（大多数练习只需要 core 包）
cd libs/core
uv sync --all-groups

# 运行 demo
uv run python ../../learning/demos/lesson01_hello_agent.py
```

## 回答风格

- 优先用**类比和图示**解释概念，再给代码
- 每个概念回答"为什么需要它"（解决什么问题）
- 如果涉及源码，指出具体文件和关键行
- 不确定时，优先查官方文档而非直接看源码猜测

## 学习节奏建议

- 每课花 45–60 分钟：读文档 15min → 动手写 30min → 可选读源码 15min
- 如果某节课卡住，先跳到下一课，不要在一节课上耗太久
- demo 代码跑不通很正常（API key、网络、版本问题），先理解逻辑即可
